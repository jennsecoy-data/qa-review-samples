# Documentation Review Sample

This sample shows how a documentation review can be organized when checking content, records, forms, or support materials for accuracy, completeness, consistency, and guideline compliance.

## Review Purpose

The purpose of this review is to compare a final document against the approved source material and confirm that all required content, formatting, and edits were applied correctly.

This type of review can be used for:

- Forms
- Records
- Reports
- Online content
- Training materials
- Customer support documentation
- Assessment or testing materials
- Internal process documents

## Review Scope

For this sample review, the following areas were checked:

- Source-to-final content match
- Requested edit verification
- Formatting consistency
- Required fields and sections
- Image captions and labels
- Online and paper version alignment
- Accessibility or alternate-format notes
- Spelling, grammar, punctuation, and readability
- Issue documentation and recommended corrections

## Sample Source Requirement

The approved source material required the final document to include:

- Updated instruction text
- Correct item number
- Matching image caption
- Required accessibility note
- Consistent heading format
- Final review status field

## Sample Review Findings

| Review Area | Finding | Status |
|---|---|---|
| Instruction Text | Final wording matches approved source | Pass |
| Item Number | Item number is correct | Pass |
| Image Caption | Caption wording matches source, but punctuation differs | Needs Review |
| Accessibility Note | Required note is missing from online version | Needs Correction |
| Heading Format | Heading spacing is inconsistent on one page | Needs Review |
| Final Review Status | Status field is present and complete | Pass |

## Detailed Review Notes

### Instruction Text

The final instruction text was compared against the approved source material. No wording differences were found.

**Status:** Pass

### Item Number

The item number in the final version matches the approved source material.

**Status:** Pass

### Image Caption

The image caption uses the correct wording, but punctuation differs from the approved version. The difference does not change the meaning, but it should be reviewed for consistency.

**Recommended action:** Confirm whether punctuation should match the approved source exactly and update if needed.

**Status:** Needs Review

### Accessibility Note

The required accessibility note is included in the paper version but missing from the online version. This creates a mismatch between formats.

**Recommended action:** Add the missing accessibility note to the online version and complete a second verification pass.

**Status:** Needs Correction

### Heading Format

One heading has extra spacing compared with the rest of the document. This appears to be a formatting consistency issue.

**Recommended action:** Adjust heading spacing to match the document style.

**Status:** Needs Review

## Summary of Issues

| Issue | Severity | Recommended Action |
|---|---|---|
| Caption punctuation differs from approved source | Low | Review and standardize punctuation |
| Accessibility note missing from online version | High | Add missing note and recheck |
| Heading spacing inconsistent | Low | Adjust formatting |

## Final Review Summary

Most of the final document matches the approved source material. The instruction text, item number, and final review status field are accurate and complete.

Three issues were found during review:

1. Caption punctuation differs from the approved source.
2. The online version is missing the required accessibility note.
3. One heading has inconsistent spacing.

The accessibility note should be corrected before final approval. The caption punctuation and heading spacing should be reviewed for consistency.

## Recommended Next Steps

- Add the missing accessibility note to the online version.
- Confirm whether caption punctuation must match the approved source exactly.
- Correct heading spacing for consistency.
- Complete a second review pass after updates.
- Document final status once corrections are verified.
